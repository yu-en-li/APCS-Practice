## 📝 刷題常用：高效 C++ 模板
在解題時，為了避免大量資料輸入輸出導致 TLE (Time Limit Exceeded)，我設定了標準的 Snippet 加速範本：

```cpp
#include <iostream>
using namespace std;

int main() {
    // 關閉輸入輸出流同步，加速 cin/cout 效率
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    
    
    return 0;
}
