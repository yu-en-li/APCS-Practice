## 💡 關鍵筆記 (Key Takeaways)

> *「理解演算法的背後邏輯，比記住程式碼更重要。」*

1. **時效性**：針對題目限制決定使用 $O(n^2)$ 還是 $O(n \log n)$ 的演算法。
2. **策略選擇**：當數據呈現特定規則時，優先考慮 Greedy 或 Two Pointers 以降低複雜度。
