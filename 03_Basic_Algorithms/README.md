# 03_Basic_Algorithms: 演算法實作與思維

本資料夾紀錄學習 APCS 基礎演算法的過程。此階段專注於解決問題的效率與邏輯優化，是從「寫出程式」到「寫出高效程式」的關鍵分水嶺。

<br>

## 📚 學習重點 (Key Concepts)

<br>

- **[01_Sorting](./01_Sorting/)**：排序演算法 (Sort) 的原理與比較，熟練 `std::sort` 的自訂排序函數。
  
<br>

- **[02_Binary_Search](./02_Binary_Search/)**：二分搜尋法 (Binary Search) 的邊界處理與單調性應用。
  
<br>

- **[03_Greedy](./03_Greedy/)**：貪婪法 (Greedy) 的決策邏輯與反例驗證。

<br>

- **[04_Brute_Force](./04_Brute_Force/)**：暴力枚舉與搜尋空間的縮減，這是解題的第一步。

<br>

- **[05_Two_Pointers](./05_Two_Pointers/)**：雙指標 (Two Pointers) 的線性掃描優化技巧。

<br>

- **[06_Math_Theory](./06_Math_Theory/)**：數論基礎 (質數判定、最大公因數、模運算)。

## 💡 關鍵筆記 (Key Takeaways)

<br>

> *「演算法的核心在於平衡時間與空間複雜度。」*

<br>

1. **優先策略**：先透過暴力枚舉確認答案可能性，再尋找規律（Greedy 或 Math）或優化搜尋效率（二分搜尋）。
2. **邊界意識**：特別是 `Binary Search` 與 `Two Pointers`，指標指向的終止條件是 Debug 的重點。
