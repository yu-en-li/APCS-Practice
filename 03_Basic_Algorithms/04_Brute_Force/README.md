# 04_Brute_Force: 暴力窮舉與全面列舉

本節探討如何透過最直覺的邏輯，有系統地列舉所有可能解來解決問題。

<br>

---

<br>

## 📌 學習目標 (Objectives)
- **基礎鞏固**：掌握巢狀迴圈控制、計數狀態與全排列的基本窮舉方法。
- **效率優化**：學會評估執行次數，理解 $O(N)$ 到 $O(N^2)$ 的時間複雜度限制。
- **邊界案例**：精確控制迴圈的初始值與終止條件，避免邊界漏算。

## 💡 解題筆記 (Key Notes)
- **核心重點**：暴力窮舉的核心是「不重不漏」，利用電腦的高速運算把所有狀況試出來。
- **常見陷阱**：當資料量過大（例如 $N \ge 10^5$）時，盲目使用雙層迴圈窮舉會導致 TLE。
- **進階思考**：找出窮舉過程中的重複計算或單調性，是後續優化成二分搜尋或雙指標的關鍵。
