## 💡 解題筆記 (Key Notes)
- **核心重點**：DP 是「帶記憶功能的枚舉」，關鍵在於如何定義狀態。
- **常見陷阱**：狀態轉移定義不明確，或是重複計算沒有妥善儲存。
- **進階思考**：DP 與圖論結合（如 DAG 最長路徑）是難題的核心考點。
