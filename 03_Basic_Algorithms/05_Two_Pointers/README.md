# 05_Two_Pointers: 雙指標演算法

本節探討如何利用兩個不同方向或同向移動的指標，在高效率的狀態下線性掃描資料。

<br>

---

<br>

## 📌 學習目標 (Objectives)
- **基礎鞏固**：理解對向指標（頭尾夾擊）與同向指標（滑動窗口初探）的運作機制。
- **效率優化**：學會將原本需要 $O(N^2)$ 的巢狀窮舉優化至 $O(N)$ 的線性時間。
- **邊界案例**：嚴密控制指標相遇（`left >= right`）或交錯時的跳出邊界條件。

## 💡 解題筆記 (Key Notes)
- **核心重點**：雙指標的核心是「利用資料的單調性（如已排序）來排除不可能的解區間」。
- **常見陷阱**：指標更新順序（`++` 或 `--`）放錯位置，或是忘記邊界檢查導致陣列索引越界（RE）。
- **進階思考**：在滑動窗口（Sliding Window）問題中，右指標負責擴展窗口，左指標負責在不滿足條件時收縮窗口。
