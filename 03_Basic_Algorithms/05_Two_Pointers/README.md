## 💡 解題筆記 (Key Notes)
- **核心重點**：回溯法的精髓在於「做選擇」 -> 「遞迴」 -> 「撤銷選擇」。
- **常見陷阱**：剪枝條件設定過於保守，導致搜尋效率依舊極低。
- **進階思考**：針對棋盤類問題（如 N-Queens），利用陣列紀錄標記以實現 $O(1)$ 剪枝檢測。
