# 03_Greedy: 貪婪法

本節學習如何在每個步驟選擇「局部最優解」，從而達成「全域最優解」。

<br>

---

<br>

## 📌 學習目標 (Objectives)
- **基礎鞏固**：練習常見貪婪情境（如排程、找零問題）。
- **效率優化**：貪婪法通常具有極佳的 $O(N \log N)$ 或 $O(N)$ 複雜度。
- **邊界案例**：證明貪婪策略的正確性，避免陷入局部最佳的假象。

## 💡 解題筆記 (Key Notes)
- **核心重點**：貪婪法需確保問題具備「貪婪選擇性質」與「最佳子結構」。
- **常見陷阱**：貪婪策略往往看起來很直覺，但若無法證明其正確性，極易導致錯誤答案。
- **進階思考**：若無法使用貪婪法，通常代表需要轉向 DP 或枚舉。
