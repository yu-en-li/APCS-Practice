## 💡 解題筆記 (Key Notes)
- **核心重點**：`std::sort` 是非穩定的，若需維持相同元素順序，請使用 `std::stable_sort`。
- **常見陷阱**：比較函式若沒遵守「嚴格弱排序 (Strict Weak Ordering)」，會導致程式崩潰。
- **進階思考**：當數據範圍小但密集時，可考慮使用計數排序 (Counting Sort) 達到 $O(N)$。
