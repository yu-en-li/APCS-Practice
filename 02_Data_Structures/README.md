# 02_Data_Structures: 資料結構實作

本資料夾紀錄學習 APCS 常見資料結構的過程。資料結構是演算法的骨架，選擇合適的結構能大幅降低時間複雜度。

## 📚 學習重點 (Key Concepts)

<br>

- **[01_Array](./01_Array/)**：靜態陣列的操作、記憶體配置與指標基礎。
  
<br>

---

<br>

## 💡 關鍵筆記 (Key Takeaways)

<br>

> *「資料結構決定演算法的上限。」*

<br>

1. **容器選擇**：頻繁增刪元素時優先考慮 `Vector`，若需要固定長度且考慮記憶體存取速度則用 `Array`。
2. **記憶體管理**：掌握物件在記憶體中的分配方式（Stack vs Heap），避免不必要的拷貝與溢位。
