# 03_String: 字串處理

本節重點在於探討字串的表示、比較與常見的字串搜尋演算法。

<br>

---

<br>

## 📌 學習目標 (Objectives)
- **基礎鞏固**：熟練字串的子字串擷取、連接與比對操作。
- **效率優化**：掌握字串 Hash 機制，快速判斷字串相等性。
- **邊界案例**：處理空字串與特殊字元（如換行符）的輸入與處理。

## 💡 解題筆記 (Key Notes)
- **核心重點**：字串在 C++ 中是 mutable 的，這與 Python 不同，需注意函式參數傳遞時的記憶體消耗。
- **常見陷阱**：隱式轉換造成的額外記憶體拷貝；使用 `stringstream` 處理複雜輸入時的效能損耗。
- **進階思考**：運用 KMP 或 Z-Algorithm 解決字串匹配問題，將時間複雜度降至線性。
