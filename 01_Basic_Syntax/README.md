## 💡 關鍵筆記 (Key Takeaways)

<br>

> *「基礎不紮實，複雜演算法往往會在邊界條件上出錯。」*

<br>

1. **I/O 加速**：在 C++ 中，務必使用 `ios_base::sync_with_stdio(0); cin.tie(0);` 來避免超時。
2. **邊界處理**：迴圈的判斷式應特別注意是否會造成「無限迴圈」或「少跑一次」的問題。
