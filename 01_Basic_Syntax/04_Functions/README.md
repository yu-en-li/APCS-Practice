## 💡 解題筆記 (Key Notes)
- **核心重點**：大物件（如 `vector`, `string`）作為參數時，務必使用「傳址呼叫 (Pass by reference)」加上 `const` 修飾詞，以避免無謂的記憶體複製。
- **常見陷阱**：區域變數遮蔽 (Shadowing) 全域變數導致資料異常；遞迴函式未設定出口導致 Stack Overflow。
- **進階思考**：利用「記憶化搜索 (Memoization)」將重複計算的函式結果暫存，顯著提升複雜遞迴的執行效率。
