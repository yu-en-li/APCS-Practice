## 💡 解題筆記 (Key Notes)
- **核心重點**：迴圈的本質是狀態轉移；確保迴圈的「不變量 (Invariant)」在每次迭代中維持正確。
- **常見陷阱**：`for` 迴圈邊界條件（例如 `<` vs `<=`）判斷錯誤導致 Off-by-one error；或是遞迴中未正確處理終止條件導致「無限迴圈」。
- **進階思考**：透過雙指標 (Two Pointers) 技巧將巢狀迴圈 ($O(n^2)$) 優化為單層迴圈 ($O(n)$)。
