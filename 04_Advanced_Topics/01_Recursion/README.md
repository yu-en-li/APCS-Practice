# 01_Graph_Basics: 圖論基礎

本節探討圖的儲存方式與基礎走訪，是深入複雜圖論演算法的基石。

<br>

## 📌 學習目標 (Objectives)
- **基礎鞏固**：熟練鄰接矩陣 (Adjacency Matrix) 與鄰接串列 (Adjacency List) 的轉換。
- **效率優化**：根據圖的密度選擇適合的儲存結構。
- **邊界案例**：處理連通塊 (Connected Components) 與斷點。

## 💡 解題筆記 (Key Notes)
- **核心重點**：DFS 適合走訪與遞迴相關問題；BFS 適合尋找無權圖的最短路徑。
- **常見陷阱**：在有環圖中進行 DFS 未標記 `visited` 導致無窮迴圈。
- **進階思考**：利用並查集 (DSU) 高效判斷圖的連通性。
