# 06_Graph_Tree: 圖論與樹狀圖演算法

本節探討非線性資料結構中，由節點 (Nodes) 與邊 (Edges) 組成的圖形結構，以及具備無環連通特性的樹狀結構之核心演算法。

<br>

---

<br>

## 📌 學習目標 (Objectives)
- **基礎鞏固**：掌握圖形的儲存方式，包括相鄰矩陣 (Adjacency Matrix) 與相鄰串列 (Adjacency List)。
- **效率優化**：在稀疏圖中棄用 $O(V^2)$ 的相鄰矩陣，改用 `vector` 實作相鄰串列以達到 $O(V+E)$ 效能。
- **邊界案例**：處理圖形中的孤立點、自環 (Self-loop)、重邊 (Multiple edges) 以及樹的根節點判定。

💡 解題筆記 (Key Notes)
- **核心重點**：樹是一種「有 $V$ 個節點且恰好有 $V-1$ 條邊的連通無向圖」；在實作圖論題時，先搞清楚給定的測資是稀疏圖還是稠密圖，從而選擇適當的資料結構。
- **常見陷阱**：在樹狀圖中使用 DFS 遍歷時，若沒有記錄父節點（Parent）是誰，程式會往回搜陷入死迴圈；另外若陣列範圍開錯會導致存取違規。
- **進階思考**：APCS 極常考察樹的直徑（Tree Diameter）以及樹狀 DP 等觀念。掌握樹的基本性質（例如任意兩點只有唯一一條路徑）能為後續的樹狀圖進階題打下穩固的地基。
