本節探討處理區間查詢與修改的高級資料結構。

## 📌 學習目標 (Objectives)
- **基礎鞏固**：理解線段樹的建立 (Build)、修改 (Update) 與查詢 (Query)。
- **效率優化**：利用 Lazy Propagation 將區間修改降至 $O(\log N)$。
- **邊界案例**：處理邊界條件與遞迴節點編號。

## 💡 解題筆記 (Key Notes)
- **核心重點**：線段樹適合處理具備結合律的區間查詢（如最大值、區間和）。
- **常見陷阱**：節點空間大小宣告不足（建議 $4N$）；懶標記 (Lazy tag) 的下放順序錯誤。
- **進階思考**：進階應用如「動態開點」或與「掃描線」結合解決幾何問題。
