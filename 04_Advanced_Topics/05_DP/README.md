## 💡 解題筆記 (Key Notes)
- **核心重點**：拓撲排序僅適用於 DAG；若存在環，則無法定義線性順序。
- **常見陷阱**：忽略圖可能存在多個拓撲序列，或是入度計算錯誤。
- **進階思考**：拓撲排序是解決動態規劃在 DAG 上狀態轉移的前提。
