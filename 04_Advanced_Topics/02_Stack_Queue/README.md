## 💡 解題筆記 (Key Notes)
- **核心重點**：DFS 用於尋找所有路徑或拓撲排序；BFS 用於層級遍歷與最短距離計算。
- **常見陷阱**：忽略圖的非連通性，導致只搜尋到了部分節點。
- **進階思考**：雙向 BFS (Bidirectional BFS) 可顯著縮短搜尋時間。
