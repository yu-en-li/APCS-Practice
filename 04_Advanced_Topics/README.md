## 💡 關鍵筆記 (Key Takeaways)

<br>

> *「進階演算法不只是寫 Code，而是對問題空間的抽象建模。」*

<br>

1. **搜尋策略**：遇到迷宮或路徑搜尋，先思考是否適用 DFS/BFS；若有重複狀態則考慮 DP 或記憶化搜尋。
2. **狀態轉移**：DP 的精髓在於找到「小問題如何推導出大問題」，寫程式前務必先用紙筆畫出狀態轉移圖。
